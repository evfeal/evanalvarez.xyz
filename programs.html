My Programs and Hardware

Software

These next entries are software that I use in my daily life, note that all software here must be fully Free (as in freedom)/Libre
and run on the GNU/Linux operating system. Another (albeit softer) requirement that I have is that the program is customizable or
"hackable" and that the source code is reasonably easy to read.

Operating System (GNU/Linux)

Void Linux is my distribution of choice, xbps is really nice and I love that you can use musl instead of glibc. Another thing about
Void is that it uses runit, so no systemd! I have tried other distributions in the past, and while Arch, Artix, and Gentoo are absolutely
fantastic, I have found that Void works best for me. At the end of the day though, it doesnt really matter what distro you use.

Window Manager

I think I have used just about every Linux window manager, they all have their pros and cons, BUT, there are a few that I have always returned
to, namely dwm, sway, and bspwm. Currently I am using dwm on my laptop and bspwm on my desktop. I think I prefer dwm by just a bit, but the
setup can be annoying. Like distributions, window managers dont really matter, dont fall for the memes about the hot new wm or distro just
use whatever works best for you.

Terminal Emulator

I am usually using st with a couple patches, it's enough for me, and its pretty fast too. If im not using st, i'll be using something like
alacritty or kitty, regarding those, alacritty is a very good terminal emulator but unfortunately does not support ligatures.
my build of st can be found on my GitHub page.

Text Editor

A text editor is a developers best friend, we use it for literally everything, as such, it is important that it is comfortable and
fast, My modified Neovim fills both of these requirements. I configure Neovim in lua, and using a wide array
of plugins like lazy.nvim, alpha, and nvim-cmp. My Neovim configuration can be found at my GitHub page.

Web Browser

Like distros and window managers, I have also gone through a ton of different web browsers, Chrome, Firefox, Ungoogled-Chromium, Brave,
etc. Out of those 4 specifically, I can at least reccomend one thing, DONT BE A CHROME NORMIE!!! Chrome has *numerous* problems regarding
Privacy, Security, and BLOAT so, so, much bloat. Firefox is a fantastic browser, and Brave is alright, though they do some shady
things with their cryptocurrency so I would personally reccomend against it. Ungoogled-Chromium is a solid choice, all of the speed of chrome without
the NSA Spyware! Personally though, I still use good old Firefox. Now that the web is being more and more dominated by chromium based browsers,
using Firefox allows you to be a member of the opposition to the increasing "chromification".
